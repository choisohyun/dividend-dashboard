# 수동 데이터 입력 가이드

## 개요

데이터 임포트 페이지에서 CSV 업로드 외에도 **수동으로 데이터를 직접 입력**할 수 있습니다.

## 사용 방법

### 1. 데이터 추가

1. **데이터 임포트** 페이지 (`/import`) 방문
2. 원하는 탭 선택 (거래/배당/입금)
3. **"+ 수동 추가"** 버튼 클릭
4. 폼에 데이터 입력
5. **"저장"** 버튼 클릭

### 2. 거래 내역 입력

**필수 필드**:
- 거래일
- 종목
- 거래 유형 (매수/매도/배당 재투자)
- 수량
- 가격

**선택 필드**:
- 수수료/세금 (기본값: 0)

**자동 계산**:
- 총 금액 = 수량 × 가격 + 수수료

**예시**:
```
거래일: 2025-01-15
종목: SCHD
유형: 매수
수량: 100
가격: 30,000
수수료: 0
─────────────────
총 금액: ₩3,000,000
```

### 3. 배당 내역 입력

**필수 필드**:
- 종목
- 지급일
- 세전 금액
- 세후 금액

**선택 필드**:
- 배당락일
- 원천징수세

**자동 계산 기능**:

#### 15% 세금 자동 계산 (해외 배당)
1. 세전 금액 입력 (예: 100,000)
2. **"15% 세금 자동 계산"** 버튼 클릭
3. 원천징수세 자동 입력 (15,000)
4. 세후 금액 자동 입력 (85,000)

#### 수동 계산
- 세전 금액과 원천징수세 입력 → 세후 자동 계산
- 세전 금액과 세후 금액 입력 → 원천징수세 자동 계산

**예시 (해외 배당)**:
```
종목: SCHD
지급일: 2025-03-25
세전 금액: 100,000
─────────────────
[15% 세금 자동 계산 클릭]
─────────────────
원천징수세: 15,000
세후 금액: 85,000
```

**예시 (국내 배당, 15.4%)**:
```
종목: KOSEF_배당
지급일: 2025-01-15
세전 금액: 10,000
원천징수세: 1,540 (수동 입력)
세후 금액: 8,460 (자동 계산)
```

### 4. 입출금 내역 입력

**필수 필드**:
- 날짜
- 금액
- 유형 (입금/출금)

**선택 필드**:
- 메모

**빠른 입력 버튼**:
- 50만원
- 100만원
- 200만원 (월 정기입금)
- 300만원

**예시 (월 정기입금)**:
```
날짜: 2025-11-02
유형: 입금
[200만원 버튼 클릭]
금액: 2,000,000 (자동 입력)
메모: 11월 정기입금
```

## 검증 규칙

### 공통 검증
- ✅ 모든 필수 필드는 입력되어야 함
- ✅ 날짜는 YYYY-MM-DD 형식
- ✅ 금액/수량은 0보다 커야 함

### 거래 검증
- ✅ 수량은 양수
- ✅ 가격은 양수
- ✅ 수수료는 0 이상

### 배당 검증
- ✅ 세후 금액 ≤ 세전 금액
- ✅ 모든 금액은 양수

### 입출금 검증
- ✅ 금액은 양수 (유형에 따라 자동으로 +/- 변환)

## 팁

### 빠른 입력
1. **같은 종목 반복 입력**:
   - 첫 거래 입력 후 저장
   - 다시 "수동 추가" 클릭
   - 종목명만 다시 입력

2. **월 정기입금**:
   - 빠른 입력 버튼 사용
   - 메모에 "X월 정기입금" 패턴 사용

3. **배당 입력**:
   - 해외 종목은 "15% 자동 계산" 사용
   - 국내 종목은 원천징수세만 입력

### 데이터 확인
- 저장 후 해당 페이지(보유현황/배당/입출금)에서 확인
- 대시보드 KPI 즉시 업데이트

### 수정/삭제
- 각 페이지 (보유현황/배당/입출금)에서 수정 가능
- 설정 페이지에서 데이터 통계 확인

## FAQ

**Q. CSV와 수동 입력 중 어떤 것을 사용해야 하나요?**
A. 데이터가 많으면 CSV, 한두 건 추가는 수동 입력이 편리합니다.

**Q. 수동으로 입력한 데이터도 백업되나요?**
A. 네, 모든 데이터가 동일하게 백업됩니다.

**Q. 잘못 입력한 데이터를 수정하려면?**
A. 해당 데이터가 표시되는 페이지(보유현황/배당/입출금)에서 수정하거나, 설정 페이지에서 삭제할 수 있습니다.

**Q. 여러 건을 한번에 입력하려면?**
A. CSV 파일로 업로드하는 것이 더 효율적입니다.
